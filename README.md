# 🃏 Big Two Pro Max | 專業大老二連線對局系統

![Version](https://img.shields.io/badge/version-3.0.0-gold)
![Node](https://img.shields.io/badge/node->=14.0-green)

這是一款採用 **黑金工業風** 設計的高級線上大老二遊戲。支援即時連線、智慧選牌以及完整的房間管理系統。

## ✨ 核心特色
- **現代化介面**：採用鈦金質感 UI 與毛玻璃效果。
- **極速對局**：支援房主重新開局，無須重新登入。
- **智慧引擎**：自動判定牌型，支援循環選牌。
- **多平台支援**：響應式設計，手機、平板、電腦皆可流暢體驗。

## 🚀 快速啟動
1. 下載並安裝 Node.js
2. `npm install`
3. `node server.js`
4. 開啟瀏覽器輸入 `http://localhost:3000`
